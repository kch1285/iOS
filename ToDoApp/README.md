# ToDoApp

**오늘 해야할 일 앱 - ToDoApp**

**ToDoApp** 은 오늘 해야할 일을 작성 및 저장하고, 완료한 일들은 체크할 수 있으며 삭제할 수 있습니다. 할 일이 많을 경우 검색을 통해 쉽게 찾을 수 있습니다.

`Realm`을 이용하여 사용자가 작성한 리스트들을 저장하고 불러올 수 있습니다.
